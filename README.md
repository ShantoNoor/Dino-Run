# Dino Run
It is a Clone of Google Chrome's T-Rex-Run Game with Good Graphics.
![Overview](Image.jpeg)

## Requirements
- SDL2
- SDL2_image
- SDL2_mixer
- SDL2_ttf

## Macos build command
```bash
./build.sh
```

## Macos run command
```bash
./bin/main
```

## Macos build and run command
```bash
./build.sh && ./bin/main
```
